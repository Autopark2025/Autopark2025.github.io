Objectifs & Mécanisme de Poursuite

Le robot Cherokey 4WD est configuré pour maintenir une cible dans son champ de vision via la caméra intelligente HuskyLens.

Explications techniques :

  • Analyse de la Bounding Box : La caméra renvoie les coordonnées (X, Y) et les dimensions (Width, Height) de l'objet appris. Le robot calcule l'erreur par rapport au centre de l'image (160 px).
  • Asservissement Proportionnel (P) : Nous utilisons deux gains distincts :
    - Kp_Rotation (1.1) : Gère la correction de cap pour recentrer l'objet latéralement.
    - Kp_Distance (0.9) : Gère l'approche. Si l'objet est trop petit, le robot avance ; s'il est trop grand, le robot recule pour garder une distance de sécurité.
  • Gestion de la Friction : L'ajout d'un MIN_PWM (45) compense la résistance mécanique des moteurs, permettant au robot d'entamer son mouvement de manière fluide sans à-coups brutaux.

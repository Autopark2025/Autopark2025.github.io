Objectifs du Système

Le Cherokey 4WD est une plateforme robotique autonome visant une efficacité maximale dans le suivi de trajectoire sur piste complexe.

Objectifs Techniques

  • Optimisation de la Vision : Exploiter la puissance de la HuskyLens pour traiter les vecteurs de ligne en temps réel.
  • Fluidité de Mouvement : Minimiser les saccades par un contrôle proportionnel précis.
  • Robustesse (Anti-perte) : Garantir une continuité de mouvement même en cas de virages à angle droit (90°) ou de brèves zones aveugles.

Fonctionnement du Code

Le code repose sur trois piliers fondamentaux :

  • Calcul d'Erreur (Proportionnel) : Le robot calcule la distance entre le centre de l'image (160 px) et le centre de la ligne détectée. Cette erreur est multipliée par le gain Kp (0.7) pour corriger la vitesse des moteurs proportionnellement.
  • Anti-Saccade : Le filtre de 80ms permet de distinguer un virage réel d'un simple bruit de capteur (reflet ou ombre), évitant au robot de pivoter inutilement.
  • Stratégie de "Mémoire de Direction" : Si la ligne disparaît, le robot se souvient de la dernière direction (`lastDirection`) pour pivoter sur lui-même dans le sens du virage manqué, permettant une récupération instantanée de la piste.
